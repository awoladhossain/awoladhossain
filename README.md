## Featured projects

### SignVision — AI Sign Language Recognition
> Real-time sign language detection system with low-latency inference

- Built with **YOLOv8 + OpenCV** — trained on a custom dataset of 10,000+ samples
- Full ML pipeline: data preprocessing → training → evaluation → deployment
- Deployed as a **containerized REST API** with GPU acceleration support
- Published findings at **Springer BIM 2025** and **IEEE COMPAS 2024**

### Node.js CI/CD Pipeline & Automation
> Fully automated build, test, and deploy workflow

- **100% reduction** in manual deployment steps via GitHub Actions
- Automated unit testing with Jest, deployment to Vercel on every push/PR
- Virtual Ubuntu environment for consistent cross-platform test execution

### Centralized Institution Management Platform *(ASG SHOP™)*
> Backend system managing complex relational data across multiple modules

- RESTful APIs with **NestJS + PostgreSQL + Prisma ORM**
- Custom **RBAC middleware** supporting multi-level user hierarchies
- Centralized error handling + **Zod** request validation across all endpoints
- CI/CD pipeline with GitHub Actions for automated build and deployment
